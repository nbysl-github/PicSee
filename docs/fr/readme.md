# Guide d’utilisation PicSee

PicSee est un visionneur d’images léger en mode « waterfall » (cascade) pour Windows. Il se concentre sur un chargement rapide et une prévisualisation fluide, avec navigation par arborescence de dossiers, gestion Favoris/Historique, tri et changement de mise en page, ainsi qu’une interface multilingue intégrée.

## 1. Démarrage rapide

1. Sélectionnez un dossier dans l’arborescence à gauche.
2. Les images du dossier s’affichent à droite en vue cascade.
3. Cliquez sur une image pour ouvrir la fenêtre d’aperçu haute qualité (zoom, déplacement, précédent/suivant, diaporama).

## 2. Barre d’outils supérieure

Principaux contrôles de gauche à droite :

*   **Recherche** : filtre en temps réel par nom de fichier.
*   **Filtre de format** : filtre par JPG/PNG/GIF/BMP/WEBP/SVG/RAW, etc.
*   **Filtre de taille** : grande / moyenne / petite.
*   **Langue** : utilisez la liste déroulante en haut à droite pour changer la langue de l’interface (persistant ; par défaut, suit la langue du système).
*   **Aide** : ouvre la documentation correspondant à la langue actuelle.

Langues intégrées :

*   简体中文
*   繁體中文
*   English
*   日本語
*   Français
*   Deutsch

## 3. Arborescence de gauche

L’arborescence contient trois nœuds racine :

### 3.1 Ce PC

*   **Bascule du thème** : cliquez sur l’icône d’ordinateur à gauche de « Ce PC » pour passer du thème clair au thème sombre.
*   **Bascule du mode de scan** : cliquez sur l’icône à droite sur la ligne « Ce PC » :
    *   `scan_single` : scanne uniquement le dossier courant.
    *   `scan_multi` : scan récursif (inclut les sous-dossiers).
*   **Parcourir disques et dossiers** : développez un disque, puis cliquez sur un dossier pour charger les images.

### 3.2 Favoris

*   **Ajouter** : clic droit sur un dossier normal, puis « Ajouter aux favoris ».
*   **Retirer** : clic droit sur un élément favori, puis « Retirer des favoris ».
*   **Tout effacer** : bouton « ... » sur la racine Favoris pour tout supprimer (confirmation requise).

### 3.3 Historique

*   **Enregistrement automatique** : les dossiers récemment visités sont sauvegardés automatiquement (jusqu’à 25).
*   **Retirer un élément** : clic droit sur un élément d’historique, puis « Retirer de l’historique ».
*   **Tout effacer** : bouton « ... » sur la racine Historique pour tout supprimer (confirmation requise).

## 4. Vue cascade (côté droit)

### 4.1 Menu contextuel (zone vide)

*   **Rafraîchir** : rescanner le dossier courant.
*   **Tri** : trier par nom / date de modification / taille, en ordre croissant ou décroissant.
*   **Disposition** : basculer entre une vue cascade verticale et horizontale.

### 4.2 Menu contextuel (sur une image)

*   **Ouvrir dans l’Explorateur** : localiser le fichier.
*   **Pivoter gauche / droite** : pivoter l’image et rafraîchir l’affichage.
*   **Copier vers / Déplacer vers** : organiser des fichiers en lot.
*   **Déplacer vers la Corbeille** : supprimer (confirmation requise).

## 5. Raccourcis du séparateur

Le séparateur central entre l’arborescence et la vue cascade permet :

*   **Direction de la mise en page** : icône du haut pour basculer vertical/horizontal.
*   **Replier/déplier le panneau** : bouton flèche pour afficher/masquer le panneau gauche.

## 6. Aperçu haute qualité

### 6.1 Souris et raccourcis

*   Molette : zoom.
*   Glisser : déplacer l’image une fois zoomée.
*   `Esc` : fermer l’aperçu.
*   `←` / `→` : précédent / suivant.
*   `Space` : lecture / pause du diaporama.
*   `+` / `-` : zoom avant / arrière.

## 7. Paramètres persistants

Les paramètres suivants persistent après redémarrage :

*   **Langue de l’interface**
*   **Favoris / Historique**
*   **Mode de scan (simple / récursif)**
