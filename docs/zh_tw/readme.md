# PicSee 使用說明

PicSee 是一款面向 Windows 的輕量級瀑布流圖片瀏覽器，專注於高效能載入與流暢預覽，支援目錄樹導航、收藏/歷史管理、排序與佈局切換，以及多語言介面切換。

## 1. 快速上手

1. 在左側目錄樹中選擇一個資料夾。
2. 右側會以瀑布流顯示該資料夾內的圖片。
3. 點擊任意圖片進入高清預覽視窗，支援縮放、拖曳、左右切換與幻燈片播放。

## 2. 頂部工具列

由左到右主要包含：

*   **搜尋**：依檔名即時篩選。
*   **格式篩選**：依 JPG/PNG/GIF/BMP/WEBP/SVG/RAW 等篩選。
*   **尺寸篩選**：依大圖/中圖/小圖篩選。
*   **語言切換**：右上角語言下拉按鈕可切換介面語言（會記住設定，預設跟隨系統語言）。
*   **使用說明**：開啟目前語言對應的說明文件。

目前內建語言：

*   简体中文
*   繁體中文
*   English
*   日本語
*   Français
*   Deutsch

## 3. 左側目錄樹

目錄樹包含三個根節點：

### 3.1 此電腦

*   **主題切換**：點擊「此電腦」左側的電腦圖示，可在深色/淺色主題間切換。
*   **掃描模式切換**：點擊「此電腦」列最右側圖示切換掃描模式：
    *   `scan_single`：僅掃描目前資料夾。
    *   `scan_multi`：遞迴掃描（包含子資料夾）。
*   **磁碟與資料夾瀏覽**：展開磁碟後，點擊具體資料夾即可載入右側圖片。

### 3.2 收藏資料夾

*   **加入收藏**：在目錄樹中對任意一般資料夾按右鍵，選擇「加入收藏」。
*   **取消收藏**：對收藏項目按右鍵，選擇「從收藏移除」。
*   **清空收藏**：點擊「收藏資料夾」根節點最右側的「...」按鈕，可一鍵清空收藏（需要確認）。

### 3.3 歷史資料夾

*   **自動記錄**：自動保存最近存取資料夾（最多 25 個）。
*   **移除記錄**：對歷史項目按右鍵，選擇「從歷史移除」。
*   **清空歷史**：點擊「歷史資料夾」根節點最右側的「...」按鈕，可一鍵清空歷史（需要確認）。

## 4. 右側瀑布流視圖

### 4.1 右鍵選單（空白處）

*   **重新整理**：重新掃描目前資料夾。
*   **排序方式**：依名稱/修改時間/大小進行升冪或降冪排序。
*   **視圖佈局**：切換縱向瀑布流或橫向瀑布流。

### 4.2 右鍵選單（圖片上）

*   **在檔案總管中開啟**：定位檔案。
*   **向左旋轉 / 向右旋轉**：旋轉圖片並刷新顯示。
*   **複製到 / 移動到**：批次整理檔案。
*   **移至資源回收筒**：刪除圖片（需要確認）。

## 5. 分隔線（Splitter）快捷按鈕

位於左右區域中間的分隔條支援：

*   **切換佈局方向**：點擊分隔條頂部圖示，在縱向/橫向佈局間切換。
*   **收起/展開左側欄**：點擊分隔條中間的箭頭按鈕，釋放更多看圖空間。

## 6. 高清預覽視窗

### 6.1 滑鼠與快捷鍵

*   滾輪：縮放圖片。
*   拖曳：放大後移動查看區域。
*   `Esc`：關閉預覽。
*   `←` / `→`：上一張 / 下一張。
*   `Space`：播放 / 暫停幻燈片。
*   `+` / `-`：放大 / 縮小。

## 7. 設定持久化

以下設定會在重啟後保留：

*   **介面語言**
*   **收藏資料夾 / 歷史資料夾**
*   **掃描模式（一級/多級）**
