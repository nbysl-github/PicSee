# PicSee ユーザーガイド

PicSee は Windows 向けの軽量なウォーターフォール（瀑布流）型画像ビューアです。高速な読み込みと滑らかなプレビューに重点を置き、フォルダーのツリー移動、Favorites/History 管理、並び替えとレイアウト切替、そしてアプリ内の多言語 UI 切替に対応します。

## 1. クイックスタート

1. 左側のツリーからフォルダーを選択します。
2. 右側に、そのフォルダー内の画像がウォーターフォール表示されます。
3. 任意の画像をクリックすると高画質プレビューを開きます（ズーム、ドラッグ、前後切替、スライドショー）。

## 2. 上部ツールバー

左から右へ主な項目：

*   **検索**：ファイル名でリアルタイムに絞り込みます。
*   **形式フィルタ**：JPG/PNG/GIF/BMP/WEBP/SVG/RAW などで絞り込みます。
*   **サイズフィルタ**：大 / 中 / 小で絞り込みます。
*   **言語**：右上の言語ドロップダウンで UI 言語を切り替えます（設定は保存され、既定はシステム言語に追従します）。
*   **ヘルプ**：現在の言語に対応する説明ドキュメントを開きます。

内蔵言語：

*   简体中文
*   繁體中文
*   English
*   日本語
*   Français
*   Deutsch

## 3. 左側ツリー

ツリーには 3 つのルートノードがあります：

### 3.1 この PC

*   **テーマ切替**：「この PC」行の左側にある PC アイコンをクリックして、ダーク/ライトテーマを切り替えます。
*   **スキャンモード切替**：「この PC」行の右端アイコンをクリックして切り替えます：
    *   `scan_single`：現在のフォルダーのみスキャンします。
    *   `scan_multi`：再帰スキャン（サブフォルダーを含む）します。
*   **ドライブ/フォルダーの参照**：ドライブを展開し、フォルダーをクリックすると右側に画像を読み込みます。

### 3.2 お気に入り

*   **追加**：通常のフォルダーを右クリックし、「お気に入りに追加」を選択します。
*   **削除**：お気に入り項目を右クリックし、「お気に入りから削除」を選択します。
*   **全削除**：お気に入りのルートにある「...」ボタンで一括削除します（確認が必要です）。

### 3.3 履歴

*   **自動記録**：最近訪れたフォルダーを自動保存します（最大 25 件）。
*   **1 件削除**：履歴項目を右クリックし、「履歴から削除」を選択します。
*   **全削除**：履歴のルートにある「...」ボタンで一括削除します（確認が必要です）。

## 4. ウォーターフォール表示（右側）

### 4.1 コンテキストメニュー（空白部分）

*   **更新**：現在のフォルダーを再スキャンします。
*   **並び替え**：名前 / 更新日時 / サイズで昇順または降順に並び替えます。
*   **レイアウト**：縦方向 / 横方向のウォーターフォールレイアウトを切り替えます。

### 4.2 コンテキストメニュー（画像上）

*   **エクスプローラーで開く**：ファイルの場所を開きます。
*   **左回転 / 右回転**：回転して表示を更新します。
*   **コピー先へ / 移動先へ**：一括でファイルを整理します。
*   **ごみ箱へ移動**：削除します（確認が必要です）。

## 5. スプリッターショートカット

左側ツリーと右側表示の間にあるスプリッターでは次ができます：

*   **レイアウト方向**：上部アイコンで縦/横レイアウトを切り替えます。
*   **サイドバーの折りたたみ**：矢印ボタンで左パネルを表示/非表示にします。

## 6. 高画質プレビュー

### 6.1 マウスとショートカット

*   マウスホイール：ズーム。
*   ドラッグ：拡大時に移動。
*   `Esc`：プレビューを閉じる。
*   `←` / `→`：前へ / 次へ。
*   `Space`：スライドショーの再生 / 一時停止。
*   `+` / `-`：拡大 / 縮小。

## 7. 永続化される設定

次の設定は再起動後も保持されます：

*   **UI 言語**
*   **お気に入り / 履歴**
*   **スキャンモード（単一 / 再帰）**
