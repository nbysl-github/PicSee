# PicSee 使用说明

PicSee 是一款面向 Windows 的轻量级瀑布流图片浏览器，专注于高性能加载与流畅预览，支持目录树导航、收藏/历史管理、排序与布局切换，以及多语言界面切换。

## 1. 快速上手

1. 在左侧目录树中选择一个文件夹。
2. 右侧会以瀑布流展示该目录下的图片。
3. 点击任意图片进入高清预览窗口，支持缩放、拖拽、左右切换与幻灯片播放。

## 2. 顶部工具栏

从左到右主要包含：

*   **搜索**：按文件名实时过滤。
*   **格式筛选**：按 JPG/PNG/GIF/BMP/WEBP/SVG/RAW 等筛选。
*   **大小筛选**：按大图/中图/小图筛选。
*   **语言切换**：右上角语言下拉按钮可切换界面语言（会记住设置，默认跟随系统语言）。
*   **使用说明**：打开当前语言对应的说明文档。

目前内置语言：

*   简体中文
*   繁體中文
*   English
*   日本語
*   Français
*   Deutsch

## 3. 左侧目录树

目录树包含三个根节点：

### 3.1 此电脑

*   **主题切换**：点击“此电脑”左侧的电脑图标，可在深色/浅色主题间切换。
*   **扫描模式切换**：点击“此电脑”行最右侧的图标切换扫描模式：
    *   `scan_single`：仅扫描当前目录。
    *   `scan_multi`：递归扫描（包含子目录）。
*   **驱动器与文件夹浏览**：展开驱动器后，点击具体目录即可加载右侧图片。

### 3.2 收藏目录

*   **添加收藏**：在目录树中对任意普通文件夹右键，选择“保存到收藏目录”。
*   **取消收藏**：对收藏条目右键，选择“从‘收藏目录’中取消收藏”。
*   **清空收藏**：点击“收藏目录”根节点最右侧的“...”按钮，可一键清空收藏目录（需要确认）。

### 3.3 历史目录

*   **自动记录**：自动保存最近访问目录（最多 25 个）。
*   **移除记录**：对历史条目右键，选择“从‘历史目录’中取消”。
*   **清空历史**：点击“历史目录”根节点最右侧的“...”按钮，可一键清空历史目录（需要确认）。

## 4. 右侧瀑布流视图

### 4.1 右键菜单（空白处）

*   **刷新**：重新扫描当前目录。
*   **排序方式**：按名称/修改时间/大小进行升序或降序。
*   **视图布局**：切换垂直瀑布流或水平瀑布流。

### 4.2 右键菜单（图片上）

*   **在资源管理器中打开**：定位文件。
*   **向左旋转 / 向右旋转**：旋转图片并刷新显示。
*   **复制到 / 移动到**：批量整理文件。
*   **移至回收站**：删除图片（需要确认）。

## 5. 分割线（Splitter）快捷按钮

位于左右区域中间的分割条支持：

*   **切换布局方向**：点击分割条顶部图标，在垂直/水平布局间切换。
*   **折叠/展开左侧栏**：点击分割条中间的箭头按钮，释放更多看图空间。

## 6. 高清预览窗口

### 6.1 鼠标与快捷键

*   滚轮：缩放图片。
*   拖拽：放大后移动查看区域。
*   `Esc`：关闭预览。
*   `←` / `→`：上一张 / 下一张。
*   `Space`：播放 / 暂停幻灯片。
*   `+` / `-`：放大 / 缩小。

## 7. 设置持久化

以下设置会在重启后保留：

*   **界面语言**
*   **收藏目录 / 历史目录**
*   **扫描模式（一级/多级）**
