# PicSee User Guide

PicSee is a lightweight waterfall-style image viewer for Windows. It focuses on fast loading and smooth preview, with folder navigation, Favorites/History management, sorting and layout switching, plus an in-app Chinese/English language toggle.

## 1. Quick Start

1. Select a folder from the left tree.
2. Images from the folder appear on the right in a waterfall view.
3. Click any image to open the high-quality preview window (zoom, drag, next/prev, slideshow).

## 2. Top Toolbar

Main controls from left to right:

*   **Search**: filter by filename in real time.
*   **Format filter**: filter by JPG/PNG/GIF/BMP/WEBP/SVG/RAW, etc.
*   **Size filter**: large / medium / small.
*   **Language**: click the “中 / En” icon to switch UI language (persisted).
*   **Help**: open the documentation file for the current language.

## 3. Left Tree

The tree contains three root nodes:

### 3.1 This PC

*   **Theme toggle**: click the computer icon on the left of “This PC” to switch dark/light theme.
*   **Scan mode toggle**: click the right-side icon on the “This PC” row:
    *   `scan_single`: scan current folder only.
    *   `scan_multi`: recursive scan (include subfolders).
*   **Browse drives and folders**: expand a drive, then click a folder to load images.

### 3.2 Favorites

*   **Add**: right-click a normal folder and choose “Add to Favorites”.
*   **Remove**: right-click a favorite item and choose “Remove from Favorites”.
*   **Clear all**: click the “...” button on the Favorites root to clear all favorites (confirmation required).

### 3.3 History

*   **Auto record**: recently visited folders are stored automatically (up to 25).
*   **Remove one**: right-click a history item and choose “Remove from History”.
*   **Clear all**: click the “...” button on the History root to clear all history (confirmation required).

## 4. Waterfall View (Right Side)

### 4.1 Context Menu (Empty Area)

*   **Refresh**: rescan current folder.
*   **Sort**: sort by name / modified time / size, ascending or descending.
*   **Layout**: switch between vertical and horizontal waterfall layout.

### 4.2 Context Menu (On an Image)

*   **Open in Explorer**: locate the file.
*   **Rotate left / right**: rotate and refresh display.
*   **Copy to / Move to**: organize files in bulk.
*   **Move to Recycle Bin**: delete (confirmation required).

## 5. Splitter Shortcuts

The middle splitter between the left tree and right view supports:

*   **Layout direction**: click the top icon to switch vertical/horizontal layout.
*   **Collapse/expand sidebar**: click the arrow button to hide/show the left panel.

## 6. High-Quality Preview

### 6.1 Mouse and Shortcuts

*   Mouse wheel: zoom.
*   Drag: pan when zoomed in.
*   `Esc`: close preview.
*   `←` / `→`: previous / next.
*   `Space`: play / pause slideshow.
*   `+` / `-`: zoom in / out.

## 7. Persisted Settings

The following settings persist across restarts:

*   **UI language**
*   **Favorites / History**
*   **Scan mode (single / recursive)**
